# Produkty

## Szablon Dziennika Tradingowego

CSV-based trading journal. Zero makr. Zero skomplikowanych formuł.

**Co otrzymujesz:**
- Plik CSV z gotowa struktura (trade log + kolumny: entry, SL, TP, R:R, emocje)
- Instrukcja importu do Google Sheets
- Gotowe formatowanie warunkowe

**Cena:** FREE - dopoki blog istnieje

<a href="/krolestwo-journal/assets/templates/trading-journal.csv" download class="btn">Pobierz szablon (CSV)</a>

### Jak uzywac
1. Pobierz plik CSV
2. Otworz Google Sheets
3. Plik > Import > Przeslij
4. Ustaw separator na przecinek
5. Gotowe - twoj dziennik jest aktywny

### Co zawiera szablon
| Kolumna | Opis |
|---------|------|
| trade_id | Twoj unikalny identyfikator |
| data | Data zawarcia transakcji |
| para | Instrument (XAUUSD, EURUSD, MNQ) |
| kierunek | Long / Short |
| entry | Cena wejscia |
| sl | Stop loss |
| tp | Take profit |
| pnl_r | Wynik w R (risk multiple) |
| outcome | Win / Loss / BE |
| emocje | Zapis stanu psychicznego |
| notatki | Co poszlo dobrze/zle |

*To narzedzie nie jest porada inwestycyjna.*
